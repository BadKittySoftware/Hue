# Hue
# Colour Picker for Linux, Windows and macOS

**Not just another HEX picker.**

HTML, RGB, CMYK, HSL, HSV and specialist designer and trade colour palettes including RAL Classic, RAL Design, British Standard BS4800, BS381C, BS5252, and NCS — all in one lightweight app.

**950 colours across 7 palette systems. 12 colour formats. One app.**

---

## What is Hue?

Hue is a professional screen colour picker built for designers, engineers, architects, and specifiers who need more than just a HEX code. Pick any colour from your screen and instantly identify it across multiple colour systems — including the professional trade and industry standards that other colour pickers simply don't have.

---

## Features

- Live screen colour capture with real-time preview and pixel-accurate magnifier
- 12 colour formats — HEX, RGB, HSL, HSV, CMYK, HTML/CSS Named, RAL Classic, RAL Design, BS4800, BS381C, BS5252, NCS
- 950 colours across 7 palette systems in the visual Colour Chart Browser
- Palette Code Search — search any RAL, BS, NCS or CSS colour by code or name
- Spectrum and Harmony tools — Complementary, Split Complementary, Triadic, Tetradic, Analogous
- Colour Adjustment — fine-tune colours with per-channel sliders (RGB, HSV, HSL, CMYK, HEX) including alpha
- Session save and restore with optional persistence across launches
- Named colour collections — assign names to captured colours
- Export — GPL (GIMP/Inkscape/Krita), CSS variables, SCSS variables, SVG, PNG, HTML
- Always on top and adjustable window opacity — see colours beneath the app as you work
- Dark, Light, and System themes
- Reassignable hotkeys — capture and undo from the keyboard
- Lightweight single executable — no Python, no dependencies, just run it

## Installation — Linux

Download the latest release, extract, and run the install script:

```bash
tar -xzf hue-1.0.0-linux-x86_64.tar.gz
cd hue-1.0.0
sudo bash install.sh
```

Then launch from any terminal:

```bash
hue
```

Or find Hue in your application menu under **Graphics**.
